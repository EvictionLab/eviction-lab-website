Maps

{{ partial "faq-open" (dict "Question" "What do the red bubbles mean?" "UniqueNumber" "100" ) }}

The red bubbles represent the eviction rate or eviction filing rate. Bigger bubbles are bigger numbers. Smaller bubbles are smaller numbers.

{{ partial "faq-close" }} {{ partial "faq-open" (dict "Question" "What does the blue mean?" "UniqueNumber" "110" ) }}

The blue represents the concentration of the census value you selected. For example, if you selected “population,” a darker blue means a higher population and a lighter blue means a lower population.

{{ partial "faq-close" }} {{ partial "faq-open" (dict "Question" "How many locations can I compare in the map?" "UniqueNumber" "111" ) }}

You can compare three locations at a time.

{{ partial "faq-close" }} {{ partial "faq-open" (dict "Question" "Why is there no bubble for my chosen location?" "UniqueNumber" "112" ) }}

For locations with no bubble, either the eviction rate for the selected year is zero, or there is no available data.

{{ partial "faq-close" }} {{ partial "faq-open" (dict "Question" "What does \"auto\" mean in the Map Level menu? " "UniqueNumber" "120" ) }}

Selecting "auto" enables you to zoom through the geographic levels (states, counties, cities, census tracts, and block groups) automatically. You can also select your desired geographic level from the same dropdown menu.
